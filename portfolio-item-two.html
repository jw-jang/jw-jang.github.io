Digital Guitar Player

Language: JAVA

The Carplus-Strong algorithm was used to replicate guitar string sounds. The Carplus-Strong algorithm takes a wavelength which is filtered to make a guitar sound. Using queues and linked lists, a text file which contains the notes and the timestamps could be converted into a dat file which plays the song. The program plucks or tics the note at the right timestamp and the played line would be dequeued.
